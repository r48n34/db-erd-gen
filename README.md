# db-erd-gen
Free DrawSQL like website but free forever  

## Features  
1. Free forever  
2. Save SQL in various DB format  
3. Easy to use / store GUI website    

## Curretn support output  
- Postgres  
- MySQL / MariaDB
- SQLite  
- knex.js  
- kysely Postgres
- kysely sqlite
