# db-erd-gen
Free DrawSQL like website but free forever  

## Features  
1. Free forever  
2. Save SQL in postgres / knex format  
3. Easy to use / store     

## Curretn support output  
- Postgres  
- MySQL / MariaDB  
- knex.js  
- kysely.js
